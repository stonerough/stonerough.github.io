What this tool handles
EZproxy → OpenAthens
  • Proxied hostnames: vendor-host.ezproxy.waikato.ac.nz → restores vendor.host
  • Login wrappers: strips /login?url= and /login?qurl=
  • Double-wrapped: login wrapper pointing at another EZproxy URL — both layers stripped
  • Google redirects: extracts the inner EZproxy URL from google.com/url?q=
  • HTML-encoded ampersands: &amp; decoded to & throughout the URL
  • Table lookup: if vendor domain matches a confirmed entry, uses the institutional entry point URL rather than the raw content URL
Plain URL → OpenAthens
  • Wraps any plain vendor URL in the OpenAthens redirector
  • Table lookup: if vendor domain matches a confirmed entry, substitutes the institutional entry point URL
  • Falls back to wrapping the input URL if vendor is not in the table
Repair broken OA link
  • Accepts a broken go.openathens.net or proxy.openathens.net URL
  • Diagnoses common failure patterns: double-encoding, wrong institution scope, missing or malformed url= parameter
  • Table lookup: if vendor domain is recognised, replaces with the confirmed institutional entry point
  • Returns a corrected URL and a plain-text explanation of what was wrong
  • Also accepts non-OA vendor URLs in this mode — treats them as needing conversion
Confirmed entry point table
  • 107 University of Waikato vendors with confirmed, tested OpenAthens URLs
  • Covers standard redirector URLs and custom SAML/non-standard entry points
  • Takes priority over mechanically constructed URLs in all modes
Skip-listed vendors
  • Some vendors cannot use OpenAthens and are flagged as SKIP rather than converted
  • Includes Westlaw (REANZ Tuakiri SSO), incompatible vendors (Times Higher Education, NBR, Journal Surf, NZ Antarctic Society), and vendors with incomplete or deferred OA setup (Academy of Management, Trade Law Guide, Pocketmags, Disability Busters)
  • The notes column explains why each was skipped
